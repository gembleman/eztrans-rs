Ehnd 사용자 사전의 특징

1. 이지트랜스의 \Ehnd 폴더 내 "UserDict" 으로 시작하는 모든 txt 파일을 사용자 사전으로 등록 가능.
　- 기존 이지트랜스 사용자 사전은 무조건 1파일로 되어 있지만, 이제는 게임별 개별 사전도 생성 가능
　- 만약 특정 사용자 사전을 사용불가능으로 만들려면 파일명 처음에 x 등을 붙여 등록 못하게 합니다.

    예시)
    UserDict_0_상용어구.txt　→　등록
    UserDict_1_캐릭터명.txt　→　등록
    UserDict_2_기타.txt　→　등록
    xUserDict_3_게임명.txt　→　등록 안함
    ....


2. 이지트랜스 \Dat 폴더의 UserDict.jk 과는 별개로 동작합니다.
　- 이지트랜스 \Dat 폴더의 UserDict.jk 는 Ehnd 설치 시 사용 불가능이 됩니다.
　- Ehnd 를 제거하면 기존에 사용하던 사전을 그대로 사용할 수 있습니다. (설치-제거가 편리해짐)

3. 사용자 사전 등록 방법
　- 사용자 사전을 등록하는 방법은 다음과 같습니다. 원문 tab 번역문 tab 0/1 tab 속성
　- 에그헤드 사용자일 경우 ezTrans 호환 사용자 사전 (Shift+Insert)의 텍스트로 저장 기능과 비슷한 형식입니다.
　- 에그헤드의 경우 속성에 관한 설정을 할 수 없었지만, Ehnd의 경우 속성 지정을 할 수 있습니다. (속성 지정은 명사만)

    원문　　　　　　　　　번역문　　　　　 상용어구 0/명사 1　　 속성
    それからというもの　　그때부터 쭉　　　0
    弐相　　　　　　　　　니쇼우　　　　　 1　　　　　　　　　　QPN


4. 현재 사용자 사전 갱신 방법
　- 사용자 사전을 수정하고 저장하면 자동적으로 갱신됩니다.
